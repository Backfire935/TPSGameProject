介绍一下项目：
一个第三人称多人射击游戏，目前通过steam会话实现联机效果，玩家可在场景中使用各种武器进行淘汰。是在Udemy平台学习[Stephen Ulibarri](https://www.udemy.com/user/stephen-ulibarri-3/)讲师的Unreal Engine 5 C++ Multiplayer Shooter课程后，自己总结课程并根据其它所学知识扩展项目内容后对此项目的介绍。
玩家拥有行走、下蹲、奔跑、瞄准在内的几种运动姿态，想要在**后续实现飞行效果**。玩家可持有手枪，冲锋枪、步枪、狙击枪、各种发射器、投掷武器以及近战武器。在场景中拥有各种增益道具可以恢复状态。想要在**后续实现场景交互**。

下面是画大饼时间：
团队战：

玩家分为两方，使用各自的武器淘汰对方拿到分数，率先达到指定分数的队伍赢得胜利。

保卫战(**未实现**)：

玩家从1-8人，在关卡中联手抵御AI敌人从地面和天空的进攻。AI敌人的优先攻击目标是基地核心，在规定的时间内基地核心未被摧毁即可获取胜利，玩家可使用场景中的各种道具、机关、陷阱、载具联手防御。你能否和队友在指定时间内守住敌人连绵不断潮水般的攻势？

(PS:此模式偏向于玩家自身和联手战斗)

塔防站(**未实现**):

玩家从1-4人，在关卡中联手抵御AI敌人从地面和天空的进攻。AI敌人的优先攻击目标是基地核心，在防守一定波数的敌人后若基地核心未被摧毁即可获取胜利，玩家可使用场景中的各种道具、机关、陷阱、载具联手防御。你能否和队友守住一波又一波的敌人？

(PS:此模式偏向于玩家放置陷阱和使用场景道具战斗)
